# Tech-Blog

## Purpose
The Tech Blog is a CMS-style blog site where developers can publish their blog posts and comment on other developers' posts as well. Follows the MVC paradigm and uses Handlebars.js.

## Built With
* Node.js
* JavaScript
* MySQL
* Handlebars
